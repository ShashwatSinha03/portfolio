# Portfolio

A high-performance editorial portfolio focused on structural clarity, engineering precision, and interactive systems.

## Essence
This project is an exploration of the intersection between rigid engineering systems and fluid design. It prioritizes architectural transparency and legible motion over generic templates.

## Tech Stack
- **Core**: Next.js 16 (App Router) + React 19
- **Motion**: [Motion](https://motion.dev/) (Framer Motion) for staggered layout transitions and micro-interactions.
- **Graphics**: [OGL](https://github.com/o-gl/ogl) (WebGL) for high-performance background shaders and 3D galleries.
- **Styles**: Modular CSS with a focus on CSS Variables and adaptive design tokens.

## Key Systems
- **Plasma Background**: An optimized WebGL fragment shader designed for atmospheric depth without GPU overhead.
- **Circular Gallery**: A curved 3D project showcase leveraging WebGL geometry and vertex manipulation.
- **True Focus**: A motion-tracked typography system that directs visual attention through dynamic blurring.
- **Dual-Theme Engine**: A robust light/dark system with smooth CSS transitions and hardware-accelerated color shifts.

## Engineering Standards
- **Optimized Performance**: Reduced shader loop complexity and geometry segments for fluid 60fps interaction on high-DPI displays.
- **Structural Legibility**: Component-based architecture with isolated logic and styling.
- **Design Tokens**: A unified 4px-grid spacing system and semantic color variables.
